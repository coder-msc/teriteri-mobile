

## 一、开始

### 相关地址链接
```sh
    后端地址：https://gitee.com/wls_javawyxawd/sp-servser
    管理端地址：
    爬虫服务地址：
```

### 前端项目初始化
node版本 18.16.0

```sh
npm install

```

### 运行

```sh
npm run dev

npm start  -- 选择的IP会更多  （推荐）
```

### 打包

```sh
npm run build
```

### 功能介绍
```sh
    登录注册、自动登录
    视频投稿、视频播放（同时在观看人数）、视频评论收藏点赞、弹幕发送
    动态发布、动态评论
    用户信息修改（头像上传裁剪，昵称等）、用户关注
    消息通知（动态评论信息、视频点赞收藏评论）
```
